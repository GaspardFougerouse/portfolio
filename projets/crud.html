Projet CRUD - Gestion des Films, Séries et Animes

Description du Projet

Ce projet est une application web permettant de gérer une base de données de films, séries et animes. Il offre des fonctionnalités complètes de CRUD (Create, Read, Update, Delete) pour les utilisateurs inscrits et connectés.

Fonctionnalités Principales

Gestion des Utilisateurs :

  • Inscription des utilisateurs avec validation par email.
  • Connexion et déconnexion sécurisées.
  • Gestion des sessions utilisateurs.

Fonctionnalités CRUD pour Films, Séries et Animes :

  • Create : Ajout de nouveaux éléments (films, séries, animes) avec des informations telles que le titre, le réalisateur, l'année, le genre, la durée, la description, le lien de l'image, le lien de la vidéo et le nombre de saisons pour les séries et animes.
  • Read : Affichage de la liste des éléments avec leurs détails.
  • Update : Modification des informations des éléments existants.
  • Delete : Suppression des éléments de la base de données.

Technologies Utilisées

Backend :

  • PHP : pour la logique de l'application et les interactions avec la base de données.
  • MySQL : pour le stockage des données.

Frontend :

  • HTML : pour la structure des pages web.
  • CSS : pour la mise en forme et le style des pages.
  • JavaScript : pour des interactions dynamiques et la manipulation du DOM.

Expérience Utilisateur

L'application est conçue pour offrir une interface utilisateur intuitive, avec une navigation simple et des formulaires clairs pour la gestion des éléments. Les utilisateurs peuvent facilement s'inscrire, se connecter et gérer leurs films, séries et animes préférés.
